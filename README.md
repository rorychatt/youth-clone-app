# YOU(th) Clone Mobile App

This repository will contain the mobile client for the YOU(th) health platform clone.

## Functionality
This application serves as the user-facing platform where individuals can:
- Track health stats from their bracelets/wearables.
- View and compare clinical laboratory results.
- Sync data with the Rust backend via REST APIs.

## Technology Stack
- **Language:** Dart
- **Framework:** Flutter (Coming Soon)

## Running Locally

*Note: The Flutter project has not been initialized yet. Once initialized, the standard execution instructions will apply.*

1. Clone the repository.
2. Install the [Flutter SDK](https://docs.flutter.dev/get-started/install).
3. Connect an emulator, simulator, or physical device.
4. From the root directory, fetch dependencies and run:
   ```bash
   flutter pub get
   flutter run
   ```
